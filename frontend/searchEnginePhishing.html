Search Engine Phishing

A search engine phishing attack involves an attacker making fake products that look attractive. When these pop up in a search engine, the target is asked to enter sensitive information before purchasing, which then goes to a hacker.

Real-Life Examples

Search for a popular website, such as Amazon.com, on a search engine and click on one of the top results. However, instead of being directed to the legitimate website, you're taken to a convincing but fake version (something like Am@zon.com) designed to mimic the original site.

How to Prevent

  1. Before clicking on a search result, look carefully at the website address to make sure it looks correct and doesn’t have strange spellings.
  2. Stick to well-known search engines like Google or Bing, which are better at blocking fake websites.
  3. Make sure the website has “https” at the start of its address and a padlock symbol before entering any personal information.
  4. Use browser tools or extensions that can warn you about unsafe websites.
